# WorkHero Landing Page

This is the landing page repository for WorkHero.

## GitHub Pages Configuration

To enable GitHub Pages:

1. Go to **Settings** → **Pages**
2. Under **Build and deployment**:
   - Source: Select `Deploy from a branch`
   - Branch: Select `main`
   - Folder: Select `/ (root)`
3. Click **Save**

## Access URL

Once configured, the site will be available at:
`https://organizationivvvv.github.io/workherolp/`

## Branch Structure

- `main` - Main branch with the latest landing page code
